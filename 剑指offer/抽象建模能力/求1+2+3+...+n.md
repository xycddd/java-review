#### 题目描述
求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）
#### 算法解析
来自牛客谈论区
java版
```
public class Solution {
    public int Sum_Solution(int n) {
        int ans = n;
        boolean t=((ans!=0) && ((ans += Sum_Solution(n - 1))!=0));
        return ans;
    }
}
```
